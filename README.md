# Bone間ウェイト転送アドオン

Blender用アドオンです。ボーン間の頂点ウェイトを一括転送・ミラーコピーすることができます。

## 機能

- **ウェイト転送**: 指定した複数のボーンペア間で頂点ウェイトを一括転送
- **転送モード**: 上書き（SET）と加算（ADD）の2モードに対応
- **左右ミラー自動検出**: `.L/.R`、`_L/_R`、`_left/_right` などの命名規則から対応ペアを自動検出
- **元/先の入れ替え**: ワンクリックでコピー元・コピー先を入れ替え
- **全頂点グループ削除**: 確認ダイアログ付きで全頂点グループを一括削除

## 対応バージョン

- Blender 5.0.0 以上

## インストール

1. 本リポジトリをZIPでダウンロード（または `git clone`）
2. Blenderを開き、`編集` → `プリファレンス` → `アドオン` → `ディスクからインストール` を選択
3. ダウンロードしたZIPファイルを選択してインストール
4. アドオン一覧で「Bone間ウェイト転送アドオン」を有効化

## 使い方

1. 3Dビューポートの右サイドバー（`N`キー）を開く
2. 「ウェイト転送」タブを選択
3. メッシュオブジェクトを選択した状態でパネルを操作

### ペアの設定

- 「設定」パネルでペア数を調整
- 各ペアの「元」にコピー元の頂点グループ、「先」にコピー先の頂点グループを指定
- 左右対称のリグには「左右ミラー自動検出」ボタンで自動設定

### 転送の実行

- 「ウェイト転送（上書き）」または「ウェイト転送（加算）」ボタンをクリック
- 確認ダイアログで「OK」を選択すると転送が実行されます
